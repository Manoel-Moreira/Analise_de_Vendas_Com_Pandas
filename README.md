# Analise de Vendas Com Pandas
Limpeza, Engenharia de atributos e análise exploratória de dados de vendas com o Pandas:

Este mini projeto, criado através do curso de Fundamentos de Linguagem Python - Do básico_ a Aplicações de IA tem  como  objetivo transformar dados transacionais brutos de um e-commerce em informações confiáveis e úteis para apoiar decisões estratégicas. Ele parte de um cenário comum em empresas em crescimento: grandes volumes de dados de vendas coletados diariamente,   mas   com   problemas   de   qualidade:   valores   ausentes,   registros   duplicados, inconsistências de tipo, erros de digitação e outliers que distorcem análises e dificultam a tomada de decisão.Ao longo do trabalho, é criada uma base fictícia que simula esses desafios e, em seguida, aplica-se  um  processo  completo  de  limpeza  e  pré-processamento  com  Pandas:  conversão  de tipos  de  dados,  tratamento  de  valores  ausentes,  remoção  de  duplicatas,  correçãode  erros  e identificação/eliminação de outliers. Depois, são realizadas etapas de engenharia de atributos, como  a  criação  de  novas  colunas  (por  exemplo,  o  valor  total  de  cada  venda)  e  uma  análise exploratória  de  dados  (EDA)  para  extrair  informações  relevantes  sobre  faturamento,  produtos mais vendidos, desempenho por categoria e evolução das vendas ao longo do tempo.Por  fim,  o  projeto  destaca  a  importância  da  visualização  de  dados  na  comunicação  de resultados,  usando  gráficos  com  Matplotlib,  Seaborn  e  Plotly  para  ilustrar  indicadores  como receita por categoria, produtos mais vendidos, tendência diária de vendas e distribuição de status de entrega. O resultado é um dataset limpo e validado, acompanhado de análises e visualizações que respondem   às   principais   perguntas   de   negócio   e   podem   suportar   ações   práticas,   como otimização de estoque, campanhas de marketing e ajustes na operação logística. E ao final, um exemplo completo de como estruturar um relatório final para o projeto.Data Science Academy
